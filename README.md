# 🌿 DietWorld Nupur - Clinical Dietician Portfolio

![DietWorld Nupur](https://img.shields.io/badge/Status-Active-success) ![React](https://img.shields.io/badge/React-18.x-61DAFB?logo=react&logoColor=white) ![Vite](https://img.shields.io/badge/Vite-5.x-646CFF?logo=vite&logoColor=white) ![TypeScript](https://img.shields.io/badge/TypeScript-5.x-3178C6?logo=typescript&logoColor=white)

A premium, high-conversion web application built for Nupur, a Certified Clinical Dietician based in Nagpur, India. This website is designed to serve as a digital storefront, highlighting her expertise, transformation stories, meal plans, and providing a frictionless consultation booking flow.

## ✨ Key Features

- **Modern & Premium UI:** Designed using a custom color palette (Forest Green & Gold) to convey health, trust, and luxury.
- **Framer Motion Animations:** Smooth scroll reveals, staggered entry animations, and satisfying hover physics on all cards and buttons to make the site feel alive.
- **Frictionless WhatsApp Integration:** The booking form compiles user responses (Name, Plan, Message) and directly opens a pre-filled WhatsApp chat with Nupur, drastically increasing conversion rates by eliminating backend friction.
- **Interactive BMI Calculator:** A built-in client utility in the "Why Us" section that offers immediate value and engagement to visitors.
- **Responsive Architecture:** Fully optimized for mobile, tablet, and desktop viewing.

## 🛠️ Technology Stack

- **Framework:** [React 18](https://reactjs.org/)
- **Build Tool:** [Vite](https://vitejs.dev/)
- **Language:** [TypeScript](https://www.typescriptlang.org/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Iconography:** [Lucide React](https://lucide.dev/)
- **Styling:** Pure Modern CSS3 (Grid, Flexbox, Variable-driven theming)

## 🚀 Getting Started

To run this project locally, follow these steps:

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. Clone the repository
```bash
git clone https://github.com/gautam0222/DietWorld_Website.git
```

2. Navigate into the directory
```bash
cd dietworld-nupur-react
```

3. Install the dependencies
```bash
npm install
```

4. Start the development server
```bash
npm run dev
```

## 📈 Marketing & Conversion Optimizations
This website was engineered with marketing psychology in mind:
- **Social Proof:** Dedicated sections for real-life transformation photos and a scrolling testimonial carousel.
- **Clear Call to Actions (CTAs):** Floating WhatsApp button and high-contrast Book Now buttons ensuring users always have a way to reach out from any point on the page.
- **Simplified Booking:** Fallback "Just Enquiring" options in the dropdown prevent users from dropping off if they aren't ready to commit to a specific plan.
